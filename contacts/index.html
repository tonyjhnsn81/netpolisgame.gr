Επικοινωνία

Η γνώμη σας είναι πολύτιμη για εμάς. Είτε είστε παίκτης που θέλει να μοιραστεί μια εμπειρία, είτε εκπρόσωπος εταιρείας που ενδιαφέρεται για συνεργασία, είμαστε εδώ για να σας ακούσουμε.

Τρόποι Επικοινωνίας

Μπορείτε να επικοινωνήσετε απευθείας με τη συντακτική μας ομάδα μέσω ηλεκτρονικού ταχυδρομείου:

  • Email: info@netpolisgame.gr

Τι μπορούμε να κάνουμε για εσάς;

Μη διστάσετε να μας στείλετε μήνυμα για:

  • Διευκρινίσεις σχετικά με τις κριτικές μας.
  • Αναφορά προβλημάτων με κάποιο από τα προτεινόμενα καζίνο.
  • Προτάσεις για νέα παιχνίδια ή παρόχους που θα θέλατε να αξιολογήσουμε.
  • Τεχνικά ζητήματα που αφορούν την ιστοσελίδα μας.

Η ομάδα υποστήριξης του NetpolisGame προσπαθεί να απαντά σε όλα τα αιτήματα εντός 48 ωρών.
